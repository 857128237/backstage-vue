# backstage-vue

基于 Vue 的 UI 组件库。

# 使用npm或yarn

推荐使用 npm 或 yarn 来安装。

``` bash
npm install backstage-vue --S
```
``` bash
yarn add backstage-vue
```
如果你的网络环境不好，你可以试试其他的注册中心和工具，比如cnpm。

# 升级最新版本

``` bash
npm update backstage-vue
```

# 如何使用
``` bash
import BackstageUI from 'backstage-vue';
import 'backstage-vue/lib/backstage-vue.css';
Vue.use(BackstageUI);
```
